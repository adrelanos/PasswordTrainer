PasswordTrainer

PasswordTrainer can help remembering new and existing passwords. Should run on any GNU/Linux distribution. Depends on bash and zenity. Small source code. Bash script. Nothing compiled. Easy to verify.

Download this project as a .zip file Download this project as a tar.gz file

Version:

Password Trainer 0.0.1

Instructions:

(Should work on Debian and Ubuntu.)

(Minor tweaks needed for other distros.)

sudo apt-get install bash zenity

cd ~

git clone https://github.com/adrelanos/PasswordTrainer.git

cd PasswordTrainer

chmod +x pwtrain

./pwtrain
